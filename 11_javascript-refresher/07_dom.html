1. Create a button in HTML that changes the background color when clicked.

2. Use prompt() to ask for your favorite color, then display it on the page.

3. Use setTimeout() to show "This message appears after 2 seconds" inside a paragraph.

In JavaScript, the setTimeout() function delays code execution for a set time. For example, inside a paragraph. This demonstrates asynchronous behavior in JavaScript and how developers can control both timing and user focus.
